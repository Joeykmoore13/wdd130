How to read Rubik's cube notation

The first thing you need to do is to pick a face to be your front and turn it so it is facing straight towards you. For this page we are going to have the green side facing directly towards us. Then we need to know what each side is marked as

The image above marks what letter goes with what side. R stands for RIGHT and corresponds to the side to the right of the one facing you. L stands for LEFT, F for FRONT, U for UP, D for DOWN, and B for BACK.

When we read a letter then we turn the corresponding side clockwise. For example if we see R then we would turn the RIGHT face clockwise like this

If there is an apostrophe after the letter then we turn that side counter-clockwise. For example if we read R' then we turn the cube like this

If there is a 2 after the letter then we turn that side twice. It doesn't matter what direction. For example if we read R2 then we turn the cube like this

Heres a list of all of the most common turns. For this tutorial the most important ones to know are the R face, F face and U face. The L and D faces are used but not as much.
